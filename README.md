# MovieMania

Movie Mania is an application to save the ratings of movies you like so you'll know if you've watched and liked the movies!

## To Run

run npm run start-dev

## Build on

React & Node, Postgres

## Database

| movies      | user     | Rating Table |
| ----------- | -------- | ------------ |
| Name        | email    | userid       |
| Description | password | movieid      |
| genre       |          | rating       |
| url         |          |              |
